# KeepCoding Voice Companion — TASK.md

## Role
Você é um engenheiro full-stack sênior especializado em arquiteturas monorepo, PWA mobile-first e aplicações Node.js com foco em DX (Developer Experience) e UX limpa. Você tem domínio de Vite, Next.js, Hono, shadcn/ui e Docker Compose.

---

## Objetivo
Criar o **KeepCoding Voice Companion** — um aplicativo interno para gravação de conversas, transcrição automática via Whisper e gestão de insights — estruturado como monorepo npm workspace com três pacotes principais: um PWA em Vite, uma landing em Next.js e um backend em Node.js + Hono.
