# Demonstrativo — Jogo da Velha com IA

## Role (Papel)
Você é um engenheiro de software front-end especialista em React, Vite e Shadcn/UI, com experiência em criar aplicações interativas e visualmente atraentes para demonstrações de capacidade de IA.

## Objective (Objetivo)
Criar um jogo da velha (Tic-Tac-Toe) totalmente funcional usando Vite + React + Shadcn/UI, que sirva como demonstração prática da capacidade da IA em desenvolver uma aplicação completa e jogável do zero.

## Context (Contexto)
- O projeto é um **demonstrativo** para apresentar a capacidade da IA em criar aplicações reais e funcionais.
- O público-alvo são pessoas que querem ver a IA em ação desenvolvendo um produto concreto.
- A stack é: **Vite** (bundler), **React** (framework), **Shadcn/UI** (componentes de interface) e **Tailwind CSS** (estilização).
- O projeto já está configurado com Vite + Shadcn/UI no ambiente local.

## Instructions (Instruções)
1. Criar o componente principal `TicTacToe` com tabuleiro 3x3 interativo.
2. Implementar a lógica do jogo: alternância de jogadores (X e O), detecção de vitória, detecção de empate e reinício de partida.
3. Exibir claramente o estado atual do jogo: de quem é a vez, quem venceu ou se houve empate.
4. Destacar visualmente as células vencedoras ao final da partida.
5. Incluir um botão "Reiniciar" para começar uma nova partida.
6. Usar componentes do Shadcn/UI (Button, Card) e Tailwind CSS para estilização moderna e responsiva.
7. O código deve ser limpo, organizado e comentado para facilitar a leitura.

## Metrics (Métricas)
- [ ] O jogo deve estar 100% funcional: vitória, empate e reinício funcionando corretamente.
- [ ] A interface deve ser responsiva e visualmente clara.
- [ ] Deve utilizar ao menos 2 componentes do Shadcn/UI.
- [ ] O código deve estar organizado em componentes reutilizáveis.
- [ ] Ao final, o jogo deve ser jogável diretamente no navegador via `vite dev`.
